# ATMMobileConnection

Учебное приложение на **C# Windows Forms**, моделирующее работу банкомата, который зависит от мобильной связи с банком.

## Описание
Программа показывает, как банкомат выполняет основные операции только при наличии связи с банковской системой. Если связь отсутствует, авторизация и банковские операции становятся недоступны.

## Реализованные возможности
- авторизация по номеру карты и PIN-коду;
- проверка баланса;
- снятие наличных;
- пополнение счета;
- просмотр истории операций;
- отображение состояния связи с банком;
- работа в режимах **"связь есть" / "связи нет"**;
- вывод сообщений об ошибках;
- простая визуализация состояния банкомата.

## Как запустить
### Через Visual Studio
1. Открыть файл `ATMMobileConnection.csproj`.
2. Запустить проект.

### Через командную строку
В папке проекта выполнить:

```powershell
dotnet run
```

## Как пользоваться
1. Запустить приложение.
2. Убедиться, что связь с банком активна.
3. Ввести номер карты и PIN-код.
4. После входа можно:
   - проверить баланс;
   - снять наличные;
   - пополнить счет;
   - открыть историю операций.
5. Кнопка **«Изменить связь»** позволяет вручную переключать состояние сети для демонстрации работы банкомата при наличии и отсутствии связи.

## Особенности работы
- При отсутствии связи невозможно:
  - войти в систему;
  - проверить баланс;
  - снять деньги;
  - пополнить счет.
- Все успешные и ошибочные действия сохраняются в истории операций.
- Для снятия наличных сумма должна быть кратна 100.

## Структура проекта
- `Forms` — окна приложения;
- `Models` — модели карты, счета, операции и состояния банкомата;
- `Services` — логика авторизации, банковских операций и связи;
- `Data` — тестовые данные;
- `Enums` — перечисления состояний.

## Назначение проекта
Приложение предназначено для учебной демонстрации объектно-ориентированного подхода при разработке WinForms-программы по теме работы банкомата при мобильной связи.
